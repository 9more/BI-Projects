## Project Overview

This Business Intelligence project analyzes retail sales and operational performance data to identify trends, evaluate profitability, and measure business efficiency across products, customers, regions, and shipping operations. The project demonstrates a complete BI workflow, from data extraction and transformation to data modeling and interactive dashboard development.

## Project Workflow

### Data Ingestion

* Retrieved transactional sales data from a SQL database.
* Validated data integrity and prepared datasets for analysis.

### Data Cleaning & Transformation

* Cleaned and standardized data using SQL and Power Query.
* Addressed missing values, data inconsistencies, and duplicate records.
* Created business-ready fields to support reporting requirements.

### Data Modeling

* Designed an optimized star schema consisting of a central fact table and supporting dimension tables.
* Established relationships to improve model performance and reporting efficiency.
* Developed DAX calculated columns and measures to support key business metrics.

### Analysis & Reporting

* Built interactive Power BI dashboards to monitor:

  * Sales performance
  * Profitability trends
  * Customer segmentation
  * Regional performance
  * Shipping and operational efficiency
* Implemented KPIs, drill-through functionality, and dynamic filtering for enhanced user insights.

## Skills Demonstrated

* SQL Data Extraction
* ETL and Data Transformation
* Star Schema Data Modeling
* DAX Measures and Calculated Columns
* Power BI Dashboard Development
* KPI Design and Performance Reporting
* Data Visualization and Storytelling
* Business Performance Analysis

## Business Value

The solution transforms raw transactional data into actionable insights, enabling stakeholders to evaluate operational efficiency, identify growth opportunities, monitor profitability, and support data-driven decision-making.

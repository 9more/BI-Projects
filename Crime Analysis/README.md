# Crime Data Analytics Dashboard

## Project Overview

This Business Intelligence (BI) project analyzes crime incident data to uncover patterns, trends, and key performance indicators that support data-driven decision-making. The project demonstrates an end-to-end analytics workflow, from data extraction and transformation to dimensional modeling, DAX development, and dashboard creation.

## Project Workflow

### 1. Data Extraction

* Connected to a SQL database to retrieve crime records and related information.
* Queried and imported data into Power BI for further analysis and reporting.

### 2. Data Cleaning & Transformation

* Cleaned and standardized data using SQL and Power Query.
* Resolved missing values, duplicate records, and inconsistent data formats.
* Created business-friendly attributes to improve data quality and usability.

### 3. Data Modeling

* Designed a dimensional model using a star schema architecture.
* Developed a central Fact Crime table containing crime incident records.
* Created supporting dimension tables, including:

  * Date Dimension
  * Location Dimension
  * Crime Type Dimension
  * Victim/Offender Dimensions (where applicable)
* Established optimized relationships to support efficient reporting and analysis.

### 4. DAX Development

* Developed DAX calculated columns to derive additional business attributes and classifications.
* Created DAX measures to calculate key performance indicators (KPIs), including crime counts, trend analysis, category breakdowns, and percentage-based metrics.
* Applied time intelligence functions to analyze crime patterns across different periods and identify year-over-year and month-over-month trends.

### 5. Dashboard Development

* Built interactive Power BI dashboards to visualize crime trends, hotspots, crime categories, and temporal patterns.
* Implemented slicers, drill-through functionality, KPI cards, and dynamic visualizations to enhance user interaction and insight generation.

## Key Skills Demonstrated

* SQL Database Querying and Data Extraction
* Data Cleaning and Transformation (ETL)
* Dimensional Data Modeling (Star Schema)
* DAX Measures and Calculated Columns
* Power BI Dashboard Development
* KPI Design and Performance Reporting
* Data Visualization and Storytelling
* Business Intelligence and Analytics
* Data Quality Management
* Analytical Problem Solving

## Business Value

The project transforms raw crime data into actionable insights through robust data modeling, DAX calculations, and interactive reporting. The dashboard enables stakeholders to identify crime trends, monitor high-risk locations, evaluate performance metrics, and make informed decisions based on reliable analytical reporting.
